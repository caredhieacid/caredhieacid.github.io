🎙️ 深度搜索生成的AI新闻播客

🗣️ 四人粤语/方言对话演示

【项目概述】 独立设计并实现的端到端智能播客生成平台。提出 "Director-Actor-Engineer" (导演-演员-后期) 三位一体生成范式， 利用腾讯混元大模型将非结构化文本重构为多角色剧本，结合零样本语音克隆技术，实现电影级音频全自动化生产。

🔹 后端架构与高并发设计

  • 基于 FastAPI + AsyncIO 构建微服务后端，设计多模态推理网关。
  • 首创 "段落级异步并发管线" (Paragraph-Level Async Pipeline)，将数千字剧本分发并行推理并在内存中流式拼接，生成效率提升 500%。
  • 实现 "熔断降级机制"：主模型超时自动切换 Edge-TTS/Llama，保障竞赛高压环境下 99.9% 可用性。

🔹 核心音频引擎与 TTS 优化

  • 深度定制 AudioSynthesizer 引擎，通过 Prompt Engineering 实现 "Dialect Injection" (方言注入)，支持四川话、粤语等方言生成。支持用户音色克隆使用自己喜欢的音色。
  • 开发自动化后期系统，利用 pydub 实现基于 Side-chaining (侧链压缩) 的自动混音算法，精准调节 BGM 音量包络，实现专业电台级过渡。

🔹 解决方案设计思维

  • 践行 "模型即服务" (MaaS) 理念，通过 Agent Workflow 串联逻辑推理与 TTS 表现力。
  • 引入 Deep Research Agent (联网搜索 + RAG)，增强模型对复杂主题深度理解，自动生成兼具深度与趣味性的对话脚本。
